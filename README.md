# 4. Análises Gráficas

Foram elaborados quatro tipos de gráficos para explorar os dados de diferentes perspectivas.

## 4.1 Gráfico de Setores (Pizza)

O gráfico de setores foi utilizado para representar a distribuição percentual das categorias presentes na base de dados.

### Principais Observações

* Foi possível identificar a participação de cada categoria no conjunto analisado.
* A visualização facilita a comparação proporcional entre os grupos.
* As categorias predominantes representam a maior parcela dos registros.

---

## 4.2 Gráfico de Barras

O gráfico de barras foi utilizado para comparar quantitativamente os valores entre as diferentes categorias.

### Principais Observações

* Evidenciou as diferenças entre os grupos analisados.
* Facilitou a identificação dos maiores e menores valores.
* Permitiu uma comparação clara entre as variáveis categóricas.

---

## 4.3 Histograma

O histograma foi utilizado para analisar a distribuição dos dados numéricos.

### Principais Observações

* Possibilitou visualizar a concentração dos valores.
* Indicou a existência de faixas de maior frequência.
* Auxiliou na identificação de possíveis assimetrias na distribuição.

---

## 4.4 Boxplot

O boxplot foi utilizado para analisar dispersão, quartis e possíveis valores discrepantes.

### Principais Observações

* Permitiu visualizar a variabilidade dos dados.
* Facilitou a identificação de outliers.
* Evidenciou a distribuição dos valores em relação à mediana.

# 5. Estatística Descritiva

Foram aplicadas medidas estatísticas para compreender melhor o comportamento dos dados.

## 5.1 Medidas de Tendência Central

As medidas de tendência central calculadas foram:

### Média

Representa o valor médio dos dados analisados.

### Mediana

Representa o valor central da distribuição.

### Moda

Representa o valor que ocorre com maior frequência.

### Análise

A comparação entre média, mediana e moda permitiu verificar o comportamento geral dos dados e possíveis assimetrias na distribuição.

---

## 5.2 Medidas de Dispersão

As medidas de dispersão calculadas foram:

### Amplitude

Diferença entre o maior e o menor valor.

### Variância

Indica o grau de dispersão em torno da média.

### Desvio Padrão

Mede o quanto os dados se afastam da média.

### Análise

Os resultados mostraram o nível de variabilidade existente nos dados, permitindo avaliar sua estabilidade e consistência.

---

## 5.3 Medidas Separatrizes

Foram calculados:

* Primeiro Quartil (Q1)
* Segundo Quartil (Q2)
* Terceiro Quartil (Q3)
* Percentis

### Análise

As medidas separatrizes permitiram dividir os dados em grupos e compreender melhor sua distribuição, identificando concentrações e dispersões ao longo da amostra.
